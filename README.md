# NeoAlbum - Interactive Flip Book Album for WordPress
www.neonotion.net

NeoAlbum is a beautiful and powerful WordPress plugin that lets you turn your images and designed pages into realistic, interactive flip-book albums. With smooth page-turning effects, it delivers an engaging reading experience similar to a physical book.
Perfect for photographers, designers, artists, digital catalogs, portfolios, and creative storytelling.
---
✨ Key Features
- **Realistic Flip-Book Experience**: Natural page-turning with mouse, touch, and keyboard support.
- **Flexible Content Support**: Works with regular images as well as text-based designs converted to images.
- **Fully Customizable**:
  - Custom fonts (Google Fonts supported)
  - Complete color control
  - Adjustable margins, padding, and page size
  - Custom book covers
  - Navigation options (arrows, thumbnails, progress)
- **Shortcode Integration**: Easily embed albums anywhere on your website using simple shortcodes.
- **Fully Responsive**: Excellent experience on desktop, tablet, and mobile.
- **Lightweight & Optimized**: Fast loading with clean code.
- **User-Friendly**: No coding required — designed for maximum creative control.
---
📥 Installation
1. Download the latest version from [GitHub](https://github.com/NeoNotion/NeoAlbum).
2. Upload the `neoalbum` folder to `/wp-content/plugins/` directory.
3. Activate **NeoAlbum** from the WordPress Plugins panel.
4. Visit **NeoAlbum → Settings** to configure global options.
